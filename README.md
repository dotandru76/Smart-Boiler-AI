# 🚿 Smart Boiler AI (Self-Learning)

**מערכת לניהול דוד חכם ל-Home Assistant עם למידת מכונה וניהול אנרגיה.**

המערכת לומדת את הרגלי המקלחת בבית, מזהה את מזג האוויר בחוץ, ומקבלת החלטות חכמות מתי לחמם מים וכמה.
כוללת מנגנון **תיקון עצמי** (Self-Correction Loop) שלומד מטעויות ידניות או מחימום מיותר.

## ✨ פיצ'רים עיקריים
* 🧠 **למידה אוטונומית:** זוכרת הרגלי שימוש ב-7 הימים האחרונים.
* 📉 **חיסכון באנרגיה:** מונעת חימום מיותר אם אין סבירות למקלחת.
* 🛡️ **הגנה מובנית:** לא מאפשרת למים לרדת מתחת לטמפרטורה קריטית.
* 🇮🇱 **מותאם לישראל:** לוח שנה יהודי (סופ"ש בשישי-שבת).

## 🚀 הוראות התקנה

1.  ודא שיש לך ב-`configuration.yaml`:
    ```yaml
    homeassistant:
      packages: !include_dir_named packages
    ```
2.  העתק את הקובץ `packages/smart_boiler.yaml` לתיקיית `packages/` ב-Home Assistant שלך.
3.  בצע בדיקת תקינות (Check Configuration) והפעל מחדש.

## ⚙️ איך זה עובד?
המערכת מחשבת "ציון דחיפות" (0-100) המורכב מ:
* היסטוריית מקלחות בשעה הנוכחית.
* מספר האנשים בבית.
* טמפרטורה בחוץ.

אם הציון עובר את **סף הרגישות**, הדוד נדלק. המערכת מעדכנת את הסף הזה אוטומטית בהתאם להצלחות וכישלונות.
